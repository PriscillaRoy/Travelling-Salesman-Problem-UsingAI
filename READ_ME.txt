READ-METhis application has a UI which takes the following parametersParameters From UserStep-1 : Input the Following parameters.1) Search Strategy : SIM- Nearest Neighbor, SOPH- Genetic Algorithm2) Cost Function: BenchMark FunctionsC1,C2,C33) Number Of Nodes : Number Of Cities4) Random Number5) Maximal Effort BoundStep-2 : Click on “Get Optimal Cost Button”.Step-3 : Output pops in a new Dialog Box.Step:-4 : Click “OK”Selecting Parameters From FileStep-1 : Click on “Choose/Browse”.Step-2 : Select the File.Format of File <Search_Strategy>,<Cost_function>,<Number_Of_Nodes>,<Maximal_Effort_Bound>Step-3 : click on “Run TSP” button.Step-4: The file will be saved in the root.( /Users/2runs.txt)--------------------------------------------------------------------------------------------------------------------------1) The /src folder contains all the.java sources files.2) 2runs.txt file has the output from the benchmark runs.3) The “Priscilla_Project1_Report_AI_Fall2017” file contains the project report.4) The .jar file is the executable.
